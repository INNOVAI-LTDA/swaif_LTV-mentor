# Frontend Mentoria Lexicon (F5)

Data: 2026-03-09

## Termos proibidos (UI)
- `paciente` / `pacientes`
- `medico` / `painel medico`
- `clinica` / `clinico`
- `jornada biologica`
- `biomarcador`
- `idade bio`
- `causa provavel` (em contexto de alerta)

## Termos permitidos (UI)
- `aluno` / `alunos`
- `mentor` / `painel do mentor`
- `mentoria` / `operacao de mentoria`
- `jornada de transformacao`
- `indicador` / `indicadores`
- `score de evolucao`
- `hipotese de bloqueio`

## Regras de aplicacao
- Manter `student/client/patient` apenas na camada tecnica de adaptacao.
- Em copy funcional de interface, usar linguagem de mentoria.
- `protocolo` pode aparecer apenas como `protocolo de mentoria` ou `metodo`.
